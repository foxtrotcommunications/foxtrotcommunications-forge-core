# Contributing to forge-core

Thank you for your interest in forge-core!

## How to Participate

### Bug Reports & Issues
Found a bug or unexpected behavior? [Open an issue](https://github.com/foxtrotcommunications/foxtrotcommunications-forge-core/issues/new) with:
- Your warehouse type (BigQuery / Snowflake / Databricks / Redshift)
- The error message or unexpected output
- A minimal reproducible example if possible

### Feature Requests & Discussion
Have an idea or want to discuss the project? Use [GitHub Discussions](https://github.com/foxtrotcommunications/foxtrotcommunications-forge-core/discussions) — we read everything there.

### Pull Requests
We are **not accepting external pull requests at this stage** while the core architecture stabilizes. This policy will be revisited for v0.2.0.

If you have a fix you'd like to share, please open an issue describing the problem and your proposed solution — we'll incorporate it if it makes sense.

## Roadmap

Things we're working toward:
- Unit test suite (`pytest`)
- CI workflow on pull requests
- Expanded Redshift support (currently beta)
- Databricks + Snowflake connector documentation
